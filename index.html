Works

  1. Interactive Device Prototype

    Designed and developed an Interactive Device using ATmega328 microcontroller with help of open-source hardware project Arduino. Different sensors were used to collect the environmental awareness. Collected data using sensor-fusion were used to take decisions. Different locomotive and signalling actuators were helping the device to interact with ourtside world. This work was Inspired by a TedEx Talk by Massimo Banzi (founder of Arduino Project).
    This talk itrigued us to think about the possibilities of Interactive Devices. The things around us, sharing common world amoung us, what if they could talk about the things they feel about their sorroundings. It's cool when a plant is intelligent enough to tweet about its well being (consider: I'm groot !).

  2. DualIso: An Algorithm for Subgraph Pattern Matching on Very Large Labeled Graphs
    Base paper: link
    Graph pattern matching is having great significance in this epoc of graph databases. Pattern matching in graph structures can be done using the property of graph isomorphism. This equally challenging and practically imaportant task is hindered by the NP-Completeness of sub-graph isomorphism. The above mentioned paper gave a simple elegent idea to reduce the space complexity and thus increasing the performance. This method uses fine pruning techniques to make this possible. Implementation of this paper was done as part of course work in NITK.
